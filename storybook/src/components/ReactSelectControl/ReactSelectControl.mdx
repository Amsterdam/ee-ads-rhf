# ReactSelectControl

The ReactSelectControl component combines the power of [React Select](https://react-select.com/) with [React Hook Form](https://react-hook-form.com/) for seamless form integration and validation.

It provides a modern, accessible, and highly customizable replacement for the native `<select>` element â€” offering features like search, async options and multi-select out of the box.

This wrapper also integrates with [Amsterdam Design System](https://designsystem.amsterdam/?path=/docs/docs-introduction--docs) styles, form labels, descriptions, and error messaging.
